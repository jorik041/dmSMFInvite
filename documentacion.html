¿Que es?

dmSMFInvite, es una aplicación desarrollada por DrvyMonkey (dm), que permite el envió y/o manejo de invitaciones para el cms SMF (Simple Machines Forums). Esta basada en PHP y requiere de MySql para poder trabajar.

dmSMFInvite, es una aplicación totalmente separada de SMF, y aunque requiera, archivos como SSI.php o el Settings.php, esta no afecta de ningún modo a SMF.

Explicación de archivos.

dmSMFInvite, contiene una simple esquema de archivos.

Archivo dmSMFInvite.php - Contiene todo el procedimiento y las funciones necesarias para que esta aplicación pueda trabajar. NO se recomienda editar, a no ser que sea muy necesario.

Archivo index.php - Al igual que _saveadmin y _invite, contiene la cara principal de la aplicación. Esta procesa la información devuelta por dmSMFInvite.php.
Si el usuario es administrador, podrá cambiar la configuración por defecto, pudiendo cambiar desde el limite de invitaciones hasta el nuevo rango que tendrán los usuarios invitados.

Archivo dmSMFInvite_config.php - Es el archivo que contiene toda la información necesaria para que la aplicación pueda funcionar. Desde los datos para la conexión a la base de datos, hasta el "diseño" que tendrá el email.

Archivo dmSMFInstall.php - Este archivo desaparecerá (o debe eliminarse) al terminar la instalación de la aplicación. Solo podrá ser ejecutado por un administrador y permite crear y popular las tablas necesarias para que la aplicación funcione. Para ello, se necesita, que el dmSMFInvite_config.php ya este configurado previamente.

Archivo dmSMFUninstall.php - Solo podrá ser ejecutado por un administrador y si previamente se ha definido una contraseña para poder ser des-instalado. Permite la eliminación de las tablas, para no dejar restos en la base de datos.

Archivo verify.php - Es el encargado de procesar los miembros invitados, haciéndoles introducir un código (normalmente este se introduce automáticamente) para poder cambiar de rango. Si el sistema de invitaciones no cambia automáticamente el rango. Este archivo no se debería usar.

Otros archivos - Otros archivos como los terminantes en .log son los encargados de guardar los registros de uso. Estos archivos no deberían ser mostrados al publico, ya que contienen información sensible sobre la aplicación y/o errores del sistema.

Puesta en marcha

Para poder utilizar esta aplicación es necesario seguir unos pasos previos.

Primero, se debe configurar el archivo dmSMFInvite_config.php con los parámetros necesarios para hacer la conexión a la base de datos. Todas la variables en dicho documento vienen explicadas (bajo comentarios). Una vez hecha esta acción, se debe proceder a abrir el archivo dmSMFInstall.php bajo previlegios de administrador del SMF (con cuenta administrador autentificada en smf). Este, procederá a crear y popular las tablas necesarias en la base de datos por defecto de SMF.

Administración

La administración o configuración, puede ser cambiada desde el index.php. Para ello un administrador del foro debe acceder como normalmente lo haría para enviar una invitación.

Des-instalación

Para des-instalar esta aplicación, primero debes haber establecido una contraseña en el archivo dmSMFInvite_config.php mayor a 4 caracteres. Esta es una medida preventiva. SI NO ES NECESARIO, NO DEFINA DICHA CONTRASEÑA, YA QUE PUEDE SUPONER UN RIESGO PARA SU INSTALACION, ademas, debes ser administrador del foro para poder continuar.

Una vez completado el procedimiento, debes ejecutar el archivo dmSMFUninstall.php y seguir los pasos necesarios. Esto hará que las tablas (y su contenido) se eliminen.
Una vez completado, podrás borrar los archivos y/o el directorio de la aplicación.
